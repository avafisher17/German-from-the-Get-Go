# German from the Get-Go

A website designed to make learning German both fun and practical through interactive quizzes and games. 

## Features

* Clickable SVG Images
* Voice integration using API
* Mobile-ready landing page
* Use of local storage to save selected avatar and assigned username

## Technologies Used

* HTML5
* CSS3
* JavaScript
* [ResponsiveVoice API](https://responsivevoice.org/)
